# Canon-C with Eclipse ThreadX

This directory will contain the integration guide for using Canon-C in
Eclipse ThreadX applications.

## Scope

This guide will cover ThreadX-specific usage patterns:

- Using Canon-C arenas alongside ThreadX byte pools
- Mapping Canon-C `Result<T, E>` to ThreadX status codes at API boundaries
- Annotating ThreadX handle lifetimes with Canon-C ownership macros
- Contract checking around ThreadX service calls
- Example applications demonstrating Canon-C + ThreadX patterns
